# DevOps Project 2
***An Encoder/Decoder application using Python Tkinter***

## Features

- Based on Caesar cipher (each character's ASCII code is shifted three positions to the right (e.g., 'A' becomes 'D', 'B' becomes 'E', etc.))
- Encryption and Decryption
- Responsive design
